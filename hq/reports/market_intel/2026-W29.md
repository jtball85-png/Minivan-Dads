**7. Intent-to-use (ITU) trademark filing mechanics and cost, for the filing-timing decision.**
As of the USPTO's 2025 fee restructuring (still in effect through 2026), there is no longer a cost discount for filing early or "placeholder"-style: there is now a single base application fee of $350 per class for all applications filed under Trademark Act Sections 1 and 44, covering use-based, intent-to-use, foreign priority, and foreign registration-based applications. In other words, an ITU filing for Class 25 costs the same $350 base fee as a use-based filing — it is not a cheaper placeholder option, just an earlier one. The difference shows up later: if you file under Section 1(b) — intent to use, before you have sold anything — you pay the base fee now and additional fees when you start using the mark: Statement of Use (SOU) — $150 per class, filed once the mark is in commerce, and a request for extension of time to file the SOU — $125 per class, filable several times, six months apart, while you get to market. Extensions can be filed for up to three years post-Notice-of-Allowance before a Statement of Use is mandatory. Longer-term, government fees run about $1,000 per class over the first ten years: the Section 8 declaration at years 5–6 is $325, and the combined Section 8 and 9 renewal at year 10 is $650. One practical cost trap worth flagging to counsel directly: the USPTO base trademark application fee is $350 per class for most electronically filed Section 1 and Section 44 applications, as long as the application meets the base filing requirements — using the USPTO's standard ID Manual language for "hats" / "t-shirts" avoids a $200/class surcharge that a custom-worded description would trigger. **This mechanical detail I can confirm; the strategic judgment call — whether to file ITU now to hold the "Minivan Dads" name in line versus waiting for a use-based filing — is a legal timing/risk decision that should go to outside counsel, not be settled from this brief alone.**
