3. **Intent-to-use (ITU) trademark filing is a real, quantifiable lower-cost placeholder path — here's the concrete mechanics and cost.** Under USPTO's current (2025 fee schedule, still in effect through 2026) system: the USPTO eliminated the old two-tier system, and there is now a single base application fee of $350 per class for all applications filed under Sections 1 and 44, covering both use-in-commerce and intent-to-use filings. For Class 25 (clothing), that's a $350 filing to lock in a priority date now, before the brand is selling anything. The catch: if you file under Section 1(b) — intent to use — you pay the base fee now and additional fees when you start using the mark: Statement of Use $150 per class, filed once the mark is in commerce; and a request for extension of time to file the SOU at $125 per class, filable several times, six months apart, while you get to market. Extensions can stretch this out for up to roughly three years total before a Statement of Use is mandatory. Longer-term, post-registration maintenance runs a Section 8 declaration at $325 per class and a Section 9 renewal at $325 per class, roughly $1,000 per class in government fees over the first ten years. Bottom line for the CEO's timing decision: an ITU filing is a cheap way to reserve "Minivan Dads" in Class 25 for ~$350 now, with several years of runway (via extensions, ~$125 each) before real product sales are required — a plausible low-cost placeholder strategy.

4. **DIY filing carries real, non-refundable error risk — this is squarely outside Market Intel's competence and needs counsel.** A Trademark Center application using free-form text costs $550 per class of goods or services, because typing your own goods/services instead of selecting from the ID Manual triggers an additional $200 per class penalty fee. Fees are also non-refundable if the application is rejected — the USPTO does not issue refunds for rejected applications, meaning one error can double your costs. I can report the mechanics; I cannot and should not judge how to word the goods/services description or assess descriptiveness risk for "Minivan Dads" — that determination needs outside trademark counsel, which is a CEO-only spend/legal-commitment decision.
