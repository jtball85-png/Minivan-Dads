The one rule that protects everything

The master is never touched. Full resolution, native ratio, borderless, flattened once, exported once. Every print size is derived from it — nothing is ever resized or resampled inside the master itself.

1 — Import (card → Lightroom)

  1. Plug in the card. Open Lightroom, go straight to Import — skip manually copying to a drive first.
  2. Choose Copy (never Move) from the card.
  3. File Handling → check Make a Second Copy To:, point it at a second drive. Instant redundancy, no extra step later.
  4. Destination → Organize: By Date.

2 — Cull & rate

  1. Turn on Auto Advance (Photo menu) first.
  2. First pass: X to reject — don't delete outright yet.
  3. Use N Survey view to compare near-duplicate burst shots side by side.
  4. Rate keepers 3–5 stars.
  5. Batch-clear: Photo › Delete Rejected Photos.

3 — Develop (Lightroom)

  1. Exposure, white balance, color, lens correction — all here.
  2. No cropping in Lightroom. Crop is decided later, in Photoshop, non‑destructively.

4 — Send to Photoshop

  1. Right‑click → Edit In › Photoshop (with Lightroom adjustments).
  2. Opens a full‑resolution copy, still linked to the catalog.

5 — Edit & crop (Photoshop)

  1. Retouch, dodge/burn, composite — normal pixel work.
  2. Need a different crop mid‑edit? Press C, crop freeform for content — never to force a print ratio.
  3. Leave Delete Cropped Pixels unchecked — stays adjustable later.
  4. Never use Image Size to shrink the file. Ever.

6 — Export the master

  1. Layer › Flatten Image.
  2. File › Export › Export As — leave Width/Height untouched.
  3. Quality 10 (max), or TIFF for true lossless.
  4. Check both Convert to sRGB and Embed Color Profile.
  5. Save to garage/prints/masters/.
  6. Keep the layered PSD too — that's your real, permanent negative.

The one rule that protects everything

File › Print — never Image › Image Size. Print scales the page, not the file. The document on disk never changes.

1 — Home test print

  1. Ctrl+P → Color Management → Photoshop Manages Colors.
  2. Printer Profile → match the exact paper loaded now (glossy and matte are different profiles).
  3. Open the OS print driver settings → turn its color management OFF (avoids double conversion — washed‑out or oversaturated prints).
  4. Position & Size → type the real target size into Height/Width: 8×10 / 11×14 / 16×20.
  5. Print. Swap paper, swap only the Printer Profile, print again.

2 — Resolution gut‑check

Check Image › Image Size before you trust the print. Long‑edge pixels needed:

Size300 dpi200 dpi floor
8×103000 px2000 px
11×144200 px2800 px
16×206000 px4000 px

Below the floor → sells smaller, or not at that size. Never upscaled silently.

3 — Green light

  1. Glossy and matte both look right at home and the size clears the floor above → ready.
  2. Master already in garage/prints/masters/ → tell Claude to run it.
  3. Pipeline derives 8×10 / 11×14 / 16×20 with white borders + a proof sheet, flags anything under‑resolved.

4 — Production sample

  1. Order the hero size as a FinerWorks sample before listing it for sale.
  2. B&W tonal neutrality (no green/magenta cast) is the thing to judge.
  3. Matches your home print's intent → approve → list.

5 — Catalog care

  1. Live catalog lives on the internal drive. Always.
  2. Backups → external SSD is fine. Catalog Settings → back up every exit while unstable, weekly once healthy.
  3. Run File › Optimize Catalog periodically.

6 — If it crashes / relocate off external

  1. Close Lightroom. In File Explorer, copy the .lrcat file + its matching …Previews.lrdata folder (or a dated backup folder) onto the internal drive.
  2. Reopen Lightroom → File › Open Catalog → select that internal‑drive copy → Open. It relaunches pointed at it.
  3. There is no "Save Catalog As" command in Lightroom Classic — the copy happens in File Explorer, not inside the app.
