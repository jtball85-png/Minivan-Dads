# The Garage

Ad hoc, ungoverned working space — CEO + Claude Code, not the board. See
`CLAUDE.md`'s "Two rooms" section for the full distinction.

Nothing here is charter/tier-governed, cost-tracked, or part of a
department's memory. `garage/research/` is where the `garage-research`
skill (`.claude/skills/garage-research/SKILL.md`) writes its findings —
draft copies of what's promoted into `hq/research/` when the CEO decides
something has earned a board handoff.
